# 🌐 Multilingual Translator - Zambia Language Edition

A beautiful, responsive web application that translates text between 30+ languages including all major Zambian languages. Built with vanilla HTML, CSS, and JavaScript.


## ✨ Features

### 🌍 Language Support
- **16 Zambian Languages** including:
  - Bemba (Chibemba) - Most widely spoken in Zambia
  - Chewa (Nyanja) - Official language
  - Tonga (Chitonga) - Southern Province
  - Lozi (Silozi) - Western Province
  - Lunda, Luvale, Kaonde - North-Western Province
  - Tumbuka, Nsenga - Eastern Province
  - Lamba, Mambwe, Bisa, Lala
  - Chokwe, Mbunda, Swahili

- **Major World Languages**:
  - English, French, Spanish, German, Italian, Portuguese
  - Japanese, Korean, Chinese, Hindi
  - Russian, Arabic

### 🎯 Core Features
- 🔍 **Auto Language Detection** - Automatically detects source language
- 🔄 **Swap Languages** - Quickly swap source and target languages
- 🎤 **Text-to-Speech** - Listen to translations with natural voices
- 📋 **One-Click Copy** - Copy text to clipboard with visual feedback
- 🗑️ **Clear All** - Reset both input and output fields
- 🌙 **Dark Mode** - Eye-friendly dark theme with persistence
- ⌨️ **Keyboard Shortcuts** - Ctrl/Cmd + Enter to translate
- 📱 **Responsive Design** - Works perfectly on mobile, tablet, and desktop

### 🚀 Performance
- **Real-time Translation** - Auto-translate as you type (debounced)
- **Loading States** - Visual feedback during translation
- **Character Counter** - 500 character limit with warnings
- **Toast Notifications** - User-friendly feedback for all actions
- **Error Handling** - Graceful error messages for API failures

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables, flexbox, and animations
- **JavaScript (ES6+)** - Async/await, fetch API, event handling
- **MyMemory Translation API** - Free, no API key required
- **Web Speech API** - Native browser text-to-speech
